# SAST-
SAST创新项目计划-南京邮电大学通达学院大学生科学与技术协会大学生创新项目孵化计划。
科协内项目编号规则

SAST:代表SAST创新项目

项目来源
RE:科研（research）
SE:自拟课题(Self-study)
OT:其他（other）


项目类别：
EH：工学硬件（含软硬结合）（Engineering hardware）
SW：软件（software）
SH：理科和人文社科（Science and Humanities and Social Sciences）

项目时限：
AY：全年（all year）
HY：半年（half year）

项目编号  根据时间而定
时间：年份+月份
顺序：当月第几个立项的项目
类别：
X:科协内部成员的项目
Y:非科协内部成员的项目
例如：1712-01-X
